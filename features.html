Features

Everything it does.

A build-and-refine studio: AI agents, your own engines, Figma-grade editing on the live page, and a design library grounding each pass. Every visual change is verified on screen.

AI agents that build

Pick how much horsepower each build gets.

Every mode edits source files and proves the result against the render — the difference is how many agents work the problem.

Solo

one fast pass

One agent builds the whole site in a single grounded pass — the fastest way from a prompt to a structured page.

Studio

clarify → build → review → fix

Asks one clarifying question when the brief is vague, builds, runs a fresh-eyed self-review, then fixes what it flagged — with an enforcer that holds the quality bar.

Swarm experimental

/flow · /multiflow

A planner maps the site; one builder per page builds them in parallel on the live canvas; a reviewer judges and builders fix the flagged pages — optionally a different AI per page.
